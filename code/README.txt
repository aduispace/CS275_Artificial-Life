README_CS275

To get started, you'll need to have Python 2.7 or Python 3.5.

1. Setting Up Environment(gym and Box2D):
	GYM
		Fetch the gym code using:
		git clone https://github.com/openai/gym
		cd gym
		pip install -e '.[all]'

	Box2D
		You can install Box2D via:
		cd gym
		pip install -e '.[box2d]'

2. Run the code:
	You can run our code via:
	python 275_project
